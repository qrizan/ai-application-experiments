# AI Application Experiments

Kumpulan eksperimen dan catatan praktis dalam membangun aplikasi AI, mencakup penggunaan OpenAI API, embeddings, LangChain, RAG, agentic system dengan LangGraph, dan best practices pengembangan sistem berbasis AI.

## Notebooks

1. [Developing AI Systems](https://colab.research.google.com/drive/1a5aUKvLVJQWwg1iXCRCISYe0piWO-gUH) - response format, retry, batching, token limits, function calling, dan best practices menggunakan OpenAI API.
2. [Embeddings AI System](https://drive.google.com/file/d/1R1FFTLPzTrOudwqZUAgP2p2MG87zvvW5/view?usp=sharing) - embeddings, similarity search, dan aplikasi berbasis representasi vektor.
3. [LangChain Chatbot Mechanics](https://drive.google.com/file/d/1nw4kPO07NUpCfr-j5Mrr7sNig3KU2Pb8/view?usp=sharing) - membangun chatbot dengan LangChain, prompt templates, memory, dan ReAct agent.
4. [RAG LangChain](https://drive.google.com/file/d/1m2gYTJc-1clAil02uZmvLzWpPr1lJuRr/view?usp=sharing) - Retrieval Augmented Generation dengan LangChain, chunking, embeddings, dan vector store.
5. [Agentic System LangGraph](https://drive.google.com/file/d/1-RoVAgt0zhdP3NGKpZR24jsbvxsRY6dE/view?usp=sharing) - membangun agentic system dengan LangGraph, StateGraph, tools, dan memory.
